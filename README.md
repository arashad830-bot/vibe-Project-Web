# Vibe Project - Web

Web app built with **React** and **TypeScript**.

## Project Structure

```
src/
├── components/
│   ├── common/
│   ├── pages/
│   └── layout/
├── pages/
├── services/
├── hooks/
├── context/
├── styles/
└── utils/
public/
tests/
```

## Tech Stack

- **Language**: TypeScript
- **UI Framework**: React 18
- **Styling**: CSS Modules / Tailwind
- **Build**: Vite
- **Package Manager**: npm

## Getting Started

1. Install Node.js 16+
2. Install dependencies: `npm install`
3. Start dev server: `npm run dev`
4. Build for production: `npm run build`
